

Parfait ! Si tu veux pousser ta branche `bada` sur ton propre dépôt `https://github.com/docteur01/test.git`, il faut juste **changer le remote** ou en ajouter un nouveau. Voici comment faire étape par étape :

---

### 1️⃣ Ajouter ou remplacer le remote

Si tu veux **remplacer l’ancien remote** :

```bash
git remote set-url origin https://github.com/docteur01/test.git
```

Si tu veux **garder l’ancien remote** et ajouter un nouveau remote appelé par exemple `myrepo` :

```bash
git remote add myrepo https://github.com/docteur01/test.git
```

---

### 2️⃣ Vérifier le remote

```bash
git remote -v
```

Tu devrais voir ton URL `https://github.com/docteur01/test.git`.

---

### 3️⃣ Pousser la branche `bada`

Si tu as remplacé `origin` :

```bash
git push -u origin bada
```

Si tu as ajouté un nouveau remote `myrepo` :

```bash
git push -u myrepo bada
```

---

### 4️⃣ Vérification sur GitHub

Va sur : [https://github.com/docteur01/test](https://github.com/docteur01/test)
Tu devrais voir ta branche `bada`.

---
